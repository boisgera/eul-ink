Make / Play / Save a Pure Tone

High-level API

>>> f = 440.0
>>> x = cos(2 * pi * f * t)
>>> x
array([ 1.        ,  0.99803566,  0.99215037, ...,  ,  0.99803566])
>>> audio.io.play(x)
>>> audio.wave.write(x, "A4.wav")
